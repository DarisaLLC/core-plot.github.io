- (instancetype) init
implementation

Initializes a newly allocated CPTPlotSymbol object.

The initialized object will have the following properties:

  • anchorPoint = (0.5, 0.5)
  • size = (5.0, 5.0)
  • symbolType = CPTPlotSymbolTypeNone
  • lineStyle = a new default line style
  • fill = nil
  • shadow = nil
  • customSymbolPath = NULL
  • usesEvenOddClipRule = NO
Returns
The initialized object.
