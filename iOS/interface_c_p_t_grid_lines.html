Detailed Description

An abstract class that draws grid lines for an axis.

Method Documentation

- (nonnull instancetype) initWithFrame: (CGRect)  newFrame
implementation

Initializes a newly allocated CPTGridLines object with the provided frame rectangle.

This is the designated initializer. The initialized layer will have the following properties:

  • axis = nil
  • major = NO
  • needsDisplayOnBoundsChange = YES
Parameters
newFrameThe frame rectangle.
Returns
The initialized CPTGridLines object.

Reimplemented from CPTLayer.
